## 1 Introduction

This document is aimed at engineers who use SIM866X series modules for development. It mainly explains how to obtain Android 11 SDK source code and related resources through repo tool. The content covers download permission application, environment preparation and source code synchronization process, helping developers quickly complete the development environment.

Through this guide, users can standardize the completion of code repository initialization and synchronization operations, access to a complete SDK source directory structure, thus laying the foundation for subsequent compilation, custom development and debugging problems.

This article is for users who are new to the SIM 866X platform or who need to re-create their development environment.

## 2 repo download code

### 2.1 repo download link request

Please contact FAE to apply for download link (download link is repo method), you need to provide ssh key.

(a) Run the following command from the Linux (recommended Ubuntu, networkable) command line to generate an SSH Key (generated by default in the`.ssh` folder of the user's home directory):

```bash
$ ssh-keygen -t rsa -C "your email"
```

After execution, the following two files will be generated:

- `~/.ssh/id_rsa`: Private key (**must be kept properly, strictly prohibited from leaking**)
- `~/.ssh/id_rsa.pub`: public key

**Please only provide the contents of the public key file`id_rsa.pub` to FAE for access to the code repository; the private key`id_rsa` must be kept securely by the customer and cannot be provided to anyone.**

(b) Wait for SIMCom to add download rights and provide download commands.

(c) Execute repo init command, refer to the following command example:

```bash
$ mkdir <APPS_ROOT>
$ cd <APPS_ROOT>
$ repo init -u ssh://username@git.simcom.com:29429/manifests -m RK356X_A11.xml --repo-url=ssh://username@git.simcom.com:29429/repo
```

(d) Start downloading code

```bash
$ repo sync
```
